📏

CONTENT 容器（最大寬度：1280px）

data-block="content"

這是 CONTENT 容器，適合用於正文內容、文章段落、表單等需要舒適閱讀寬度的區塊。 最大寬度限制在 1280px（80rem），左右自動置中，確保長文閱讀時視線不會過度左右移動。

  • ✅ 最大寬度：1280px（80rem）
  • ✅ 左右自動置中（margin: 0 auto）
  • ✅ 響應式內距：手機 1rem / 平板 2rem / 桌面 1.5rem
  • ✅ 適合：文章內容、FAQ、表單、說明文字

使用範例：關於我們、產品說明、服務條款、部落格文章等需要長文閱讀的頁面。

🖼️

VISUAL 容器（最大寬度：1440px）

data-block="visual"

這是 VISUAL 容器，適合用於圖片展示、產品網格、卡片列表等視覺化內容。 最大寬度 1440px（90rem），比 CONTENT 寬 160px，可以容納更多的多欄內容。

🌐

BLEED 容器（100% 寬度）

data-block="bleed"

這是 BLEED 容器，適合用於滿版背景、Hero 區塊、大型橫幅等需要全寬設計的區塊。 寬度為 100%（無最大寬度限制），通常內部會嵌套 CONTENT 或 VISUAL 容器來包裹實際內容。

💡 設計技巧

注意：這個紫色區塊的背景色延伸到螢幕邊緣（BLEED 容器特性）， 但文字內容被包在內層的 data-block="content" 中（1280px）， 這是常見的組合用法，稱為「滿版背景 + 內容限寬」。

  • ✅ 寬度：100%（無最大寬度限制）
  • ✅ 通常搭配背景色或背景圖使用
  • ✅ 內部嵌套 CONTENT 或 VISUAL 來限制內容寬度
  • ✅ 適合：Hero 區塊、CTA 橫幅、Footer、滿版圖片區

使用範例：首頁 Hero、促銷橫幅、頁尾資訊、全寬圖片展示、CTA 區塊等
